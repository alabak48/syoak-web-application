# SYOAK Web Application

## ```src``` folder structure
```
├── assets                       #Folder for storing images
├── components                   #Folder for storing React components
   ├── achievements              #Achievement container
      ├── Achievements.css       #Achievements component stylesheet
      ├── Achievements.js        #Achievements component
   ├── animation                 #Animations from React Spring animation library
      ├── FadeInEffect.js        #React Spring animation for fade in effect
      ├── SlideUpEffect.js       #React Spring animation for slide up effect
   ├── blog                      #Blog container
      ├── Blog.css               #Blog component stylesheet
      ├── Achievements.js        #Blog component
   ├── buttonnavigation          #Container with hover button navigation
      ├── ButtonNavigation.css   #ButtonNavigation component stylesheet
      ├── ButtonNavigation.js    #ButtonNavigation component
   ├── carousel                  #Carousel container
      ├── Carousel.css           #Carousel component stylesheet
      ├── Carousel.js            #Carousel component
   ├── footer                    #Footer container
      ├── Footer.css             #Footer component stylesheet
      ├── Footer.js              #Footer component
   ├── form                      #Form and location container
      ├── Form.css               #Form component stylesheet
      ├── Form.js                #Form component
   ├── herocontainer             #Main hero container
      ├── HeroContainer.css      #Hero component stylesheet
      ├── HeroContainer.js       #Hero component
   ├── navbar                    #Navigation bar container
      ├── NavigationBar.css      #Navigation bar component stylesheet
      ├── NavigationBar.js       #Navigation bar component
├── pages
   ├── HomePage.js              #Rendering all components
├── styles
   ├── styles.css                  #PostCSS file
   ├── variables.css               #CSS Color Variables
App.js #Rendering Home page
index.js #Root render
styles.css #Importing all CSS files for PostCSS and global CSS file
```
